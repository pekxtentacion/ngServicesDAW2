Bienvenido/a a mi práctica de DAW2

En esta práctica usaremos los serivicios y el enrutamiento en Angular. Esta SPA (Single Page Application), o aplicación de página única, es una aplicación web o es un sitio web que cabe en una sola página con el propósito de dar una experiencia más fluida a los usuarios, como si fuera una aplicación de escritorio.

La aplicación se está compuesta por distintos componentes como el Home, Productos, Carrito y Contacto que se enrutan con el propósito de dar mas fluidez a la hora de navegar, sin tener que cargar cada página desde el servidor.
